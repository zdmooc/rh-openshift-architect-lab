# Lab 03 – File / Block storage avec ODF

Objectif : créer des PVC File (CephFS) et Block (RBD), et les consommer dans
des applications simples.

Fichiers :
- `pvc-samples.yaml` : PVC File et Block.
- `deployment-cephfs.yaml` : app utilisant un PVC CephFS.
- `deployment-rbd.yaml` : app utilisant un PVC RBD.
- `storageclass-custom.yaml` : exemple de StorageClass custom.
