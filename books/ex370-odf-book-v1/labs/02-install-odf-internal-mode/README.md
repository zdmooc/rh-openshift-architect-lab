# Lab 02 – Installation ODF en Internal Mode

Objectif : installer ODF en Internal Mode et vérifier que la plateforme de stockage
est opérationnelle.

Étapes :
1. Identifier et labelliser les nœuds de stockage.
2. Installer l'operator ODF.
3. Créer le StorageCluster.
4. Vérifier la santé du cluster ODF et les StorageClass.
